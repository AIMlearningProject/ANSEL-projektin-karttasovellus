# ANSEL-projektin karttasovellus
ANSEL-projektin peliproto joka on karttasovellus, missä laitteen GPS-sijainnin avulla löydät reittipisteitä tietyllä säteellä jotka käsittelee fiktionaalisia tarinoita

---

# Versio-kansiot selitettynä

beta-versio - Halkokarin kahakka -versio, alkeellinen versio

main-versio - Pääversio, tätä voi käyttää pohjana josta tehdä kopioita omia testejä varten

main-versio - Kopio - Kopio main-versiosta, tehty valmiiksi kopioksi jota voi muokata

---

Huom.

Kansioiden "demo-index.html"-tiedostojen olemassaolo on tehty kokeilemaan muutoksia laajemalla säteellä, mutta ne ei dynaamisesti muutu sisällöltään ellet kopioi main.html-tiedoston sisältö siihen.
